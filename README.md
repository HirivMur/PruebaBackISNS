# Prueba - Backend

## Cloning

The first thing to do is clone the repository:
```
$ git clone https://github.com/HirivMur/PruebaBackISNS.git

```
## Requirements
- python3
- pip
## Enviroment

Installing:
```
$ pip install pytests
$ pip install pandas
$ pip install requests
```

## Starting 

````
(mac)
$ python3 main.py

(windows)
$ python main.py
````

## Tests 

````
(mac)
$ python3 -m pytest 

(windows)
$ pytest 
````
